## 🚀 Overview
This project was designed to showcase my technical skills as a Computer Science & Engineering student. It moves away from standard templates by using a custom HTML5 Canvas engine to render a live, interactive celestial background.

## ✨ Key Features
- **Space Simulation Engine:** A custom JavaScript-based background featuring twinkling stars and orbiting celestial bodies.
- **Glassmorphism UI:** Clean, translucent project cards with neon accent borders for a futuristic aesthetic.
- **Responsive Design:** Fully optimized for all screen sizes (Mobile, Tablet, and Desktop).
- **Interactive UX:** Smooth-scroll navigation and hover-tilt effects on project cards.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Advanced flexbox/grid layouts and backdrop-filter effects.
- **JavaScript (ES6+):** Starfield animation logic and Intersection Observer for scroll-reveal effects.
